## Toolbar 이슈
### 🤯 이슈
Toolbar를 사용하면 앱이 실행되지 않는 오류

### 💦 해결
액션바는 액티비티의 구성요소로 자동으로 출력이 되고 툴바는 개발자가 레이아웃에 직접 뷰를 만들어 주게 된다. 
해당 오류가 발생한 것은 액션바와 툴바가 같은 위치에서 같은 목적으로 사용되기때문에 둘 중에 하나만 사용할 수 있기 때문이다. 
액션바가 나와야할 자리에 툴바도 나오려고 하니 문제가 발생한 것이다. 
이처럼 툴바를 사용하기 위해서는 액션바가 출력되지 않도록 설정해주어야한다. 

```xml
<style name="Theme.BookCheck" parent="Theme.MaterialComponents.DayNight.NoActionBar">
```

### 💥 배운 것
- 액션바와 툴바의 차이  
    - 액션바 : 액티비티의 구성요소로 자동으로 출력
    - 툴바 : 개발자가 직접 만드는 뷰
